Here's the documentation and tutorials for various topics about:
- mathematics: category theory, algebra, etc.
- computer science: compilers, os kernels, etc.

## Structure
For mathematics, the articles are written as a typical math textbook style, namely definitions, theorems, proofs, and examples, and also motivations and backgrounds before definitions.

For computer science, the articles are written as:
- backgrounds
- designs
- implementations

## learning progress
The content for each topic/page takes 1 or 2 months to learn, and we try to fit it into a single page. Please click the sidebar to visit each topic.
